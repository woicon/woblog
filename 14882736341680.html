git远程操作

2016/1/28 分类   code

git分支操作
git查看本地分支，带星号的为当前开发的分支。

$git branch -a //查看所有分支      
master
* newblog

$git push origin :master;//删除远程master分支
$git branch -D newbranch;//删除本地master分支
$git checkout newbranch//切换分支
